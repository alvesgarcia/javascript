Usando window.alert( )

Você pode usar uma caixa de alerta para exibir dados

Você pode optar por não utilizar a palavra-chave window. Em JavaScript, o objeto window é o objeto de escopo global. Isso significa que variáveis, propriedades e métodos, por padrão, pertencem ao objeto window. Isso também significa que especificar a palavra-chave window é opcional.
